#VideoJuego con SFML: Bomb Storm
Bomb storm es un videojuego de reflejos que nos obliga a sobrevivir a una lluvia de misiles que deberemos evitar para no perder el juego. Esta desarrollado en el lenguaje c++, con el uso de herramientas como Cmake, que se utiliza para definir los procesos de compilaci√≥n y proporcionar la configuracion para utilizar nuestra biblioteca grafica, y tambien SFML, que es una biblioteca que nos permmite implementar elementos multimedia.
